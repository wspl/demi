# @demi/provider-anthropic-api

A Demi provider for the Anthropic Messages API. Exposes
`createAnthropicApiProvider()`.

```ts
import { createAnthropicApiProvider } from '@demi/provider-anthropic-api'
```

Implements the [`@demi/provider`](../provider/README.md) contract. Part of
[Demi](../../README.md). Apache-2.0.
